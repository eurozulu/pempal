# Pempal
### X509 management tool
pempal is a tool to help you manage your x509 certificates and the resources related to them,  
Certificates, Certificate requests, Private & public keys and Certificate revokation lists.  

It is designed as a command line tool, to be used interactively or within scripts.  
  
#### Features
- Supports common file formats, der, pem, pkcs12, pkcs8
- Create, Sign, Revoke all resources for your certificates  
- Find resources based on property values or type  
- Display resources from and to any common format
- Template feature allows pre configuration of common resource.
